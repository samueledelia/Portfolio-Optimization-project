This repository encompasses three primary scripts: Run1_group10, Run2_group10, and Run3_ETF_analysis. Each script is tailored to execute distinct tasks within the project:
-Run1_group10: This script comprises the code relevant to Part A and Part B of Exam C.
-Run2: Used for backtesting various portfolios with rebalancing. Note that this file will take around 10 min to run.
-Run3_ETF_analysis: Focuses on the analysis of a portfolio based on iShares S&P 100 ETF (OEF) used to make a real world comparison.
